# go-react-redux-starter

### Requirements
* Node.js (8+)
* Go (1.8+)

### Installation
`go get github.com/wangsongiam/go-react-redux-starter`
`yarn`

### Scripts
* `yarn dev-webpack`: run frontend with webpack
* `yarn dev-server`: run backend with go-gin
